Why Daytona

Daytona makes
parallel sandboxes trivial.

🔒
True IsolationEach cauldron is a separate environment. Zero port conflicts, zero shared state.
🗑️
Auto-cleanupautoDeleteInterval: 1440 — sandboxes self-destruct after 24h.

File Upload APIsandbox.fs.uploadFile() — push AI-generated HTML directly.
⚙️
Process APIsandbox.process.executeSessionCommand() — start servers instantly.
// All it takes to spin up a sandbox
const sandbox = await daytona.create({
  language: "typescript",
  autoStopInterval: 30,
  autoDeleteInterval: 1440,
});

// Upload AI-generated HTML
await sandbox.fs.uploadFile(
  Buffer.from(html, "utf-8"),
  "index.html"
);

// Start the preview server
await sandbox.process.executeSessionCommand(
  sessionId, {
    command: "python3 -m http.server 3000",
    runAsync: true,
  }
);
